# suelliott77.github.io

Personal portfolio site for Sutton Elliott, hosted on GitHub Pages.

**Live site:** [suelliott77.github.io](https://suelliott77.github.io)

## Pages

| File | Description |
|------|-------------|
| `index.html` | Home / landing page |
| `resume_projects.html` | Professional and resume-worthy projects |
| `personal_project.html` | Personal experiments and side builds |
| `schooling.html` | Education background |
| `leadership.html` | Leadership experience |

## Structure

```
├── index.html
├── resume_projects.html
├── personal_project.html
├── schooling.html
├── leadership.html
├── style.css          # Global styles
├── animations.js      # Scroll and entrance animations
├── theme.js           # Theme utilities
├── carousel.js        # Image carousel logic
└── images/            # Project screenshots, backgrounds, resume PDF
```

## Deployment

Automatically deployed via GitHub Pages on every push to `main`. No build step required — all files are plain HTML, CSS, and JavaScript.
