TI-84 Calculator (Recreated)

Python, Mathematical Systems, Input Parsing

Developed a functional recreation of a TI-84 style calculator, focusing on accurate mathematical evaluation and reliable user input handling. This project emphasized translating mathematical rules into program logic and managing calculator state.

  • Implemented arithmetic operations and order-of-operations logic
  • Designed a state-based input system to mimic calculator behavior
  • Handled edge cases such as invalid input and chained expressions
